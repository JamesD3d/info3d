<Tabs.Tab key="Blender">
<>
[Visit Blender](https://www.blender.org/)

**Blender** is a full-featured, open-source 3D creation suite. It supports modeling, sculpting, animation, rendering, and more.

- Desktop only (PC/Mac/Linux)
- Powerful sculpting brushes and modifiers
- Industry-grade pipeline support
</>
</Tabs.Tab>

<Tabs.Tab key="SculptGL">
<>
[Visit SculptGL](https://stephaneginier.com/sculptgl/)

**SculptGL** is a lightweight, browser-based 3D sculpting tool perfect for quick models or learning the basics.

- Runs directly in browser
- No installation required
- Supports basic brushes and mesh manipulation
</>
</Tabs.Tab>

<Tabs.Tab key="Nomad Sculpt">
<>
[Visit Nomad Sculpt](https://nomadsculpt.com/)

**Nomad Sculpt** is a mobile app for sculpting with a polished interface and advanced tools.

- App only (iOS/Android)
- Great for tablets
- Touch-optimized sculpting experience
</>
</Tabs.Tab>

<Tabs.Tab key="Womp">
<>
[Visit Womp](https://womp.com/)

**Womp** is a browser-based collaborative 3D modeling tool with a clean interface and live updates.

- Cloud-based platform
- Real-time collaboration
- Beginner-friendly for organic modeling
</>
</Tabs.Tab>

<Tabs.Tab key="3D Slash">
<>
[Visit 3D Slash](https://www.3dslash.net/index.php)

**3D Slash** offers a unique, block-based approach to 3D modeling. It's especially engaging for beginners and kids.

- Browser-based and downloadable
- Gamified voxel-style interface
- Educational and intuitive
</>
</Tabs.Tab>

</Tabs>
