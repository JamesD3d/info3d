# Images  - Tools to Use Image formats to make 3D Modles 


<Tabs items={[
  'GIMP',
  'HueForge',
  'Inkscape'
]}>

<Tabs.Tab key="GIMP">
<>
[GIMP](https://www.gimp.org/)
![GIMP.webp](https://cdn.jsdelivr.net/gh/JamesD3d/info3dimages@main/i/Other_Useful/GIMP.webp)


**GIMP** (GNU Image Manipulation Program) is a free and open-source image editor used for photo retouching, image composition, and graphic design.
</>
</Tabs.Tab>

<Tabs.Tab key="HueForge">
<>
[HueForge](https://shop.thehueforge.com/)

**HueForge** is a specialized tool used to convert images into color-coded lithophanes and multi-color 3D printable models.
</>
</Tabs.Tab>

<Tabs.Tab key="Inkscape">
<>
[Inkscape](https://inkscape.org/)
![INKSCAPES.webp](https://cdn.jsdelivr.net/gh/JamesD3d/info3dimages@main/i/Other_Useful/INKSCAPES.webp)


**Inkscape** is a powerful open-source vector graphics editor ideal for creating SVGs, logos, and illustrations, often used in laser cutting and CNC workflows.
</>
</Tabs.Tab>
